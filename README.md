# Data Center Cooling – Projet PyCATSHOO

Ce projet concerne l’étude et la modélisation d’un **système de refroidissement de centre de données** à l’aide de **PyCATSHOO**.

Le code implémente un modèle de sûreté de fonctionnement permettant de simuler le comportement du système de refroidissement et d’analyser l’impact de conditions dégradées (pannes, contraintes climatiques...) sur la capacité de refroidissement de la salle informatique.
